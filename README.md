# 🏏 Interactive IPL 2026 Analytics Dashboard

## 📌 Project Overview

The IPL 2026 Analytics Dashboard is an interactive Power BI project developed to analyze team performance, player statistics, venue trends, and match insights from the Indian Premier League (IPL) 2026 season.

This dashboard transforms raw cricket data into actionable insights through data modeling, DAX calculations, interactive filters, and advanced visualizations.

---

## 🎯 Objectives

- Analyze overall IPL 2026 season performance.
- Identify top-performing teams and players.
- Compare batting, bowling, and fielding contributions.
- Evaluate venue-wise scoring trends.
- Study toss impact on match outcomes.
- Create an interactive dashboard for cricket analytics.

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- CSV Dataset
- Data Modeling & Relationships
- Interactive Visualizations

## 📈 Key Insights

### Team Performance

- Punjab Kings lead the points table with the highest points tally.
- Royal Challengers Bengaluru have the best Net Run Rate.
- Rajasthan Royals and Sunrisers Hyderabad remain strong playoff contenders.

### Batting Insights

- Abhishek Sharma leads the Orange Cap race.
- Virat Kohli and KL Rahul continue to be among the top run scorers.
- Boundary analysis highlights the most aggressive batters of the season.

### Bowling Insights

- Anshul Kamboj and Bhuvneshwar Kumar are among the leading wicket-takers.
- Jofra Archer leads the Pressure Creators analysis with the highest dot balls.

### Venue Insights

- Sawai Mansingh Stadium has produced the highest average first-innings scores.
- Teams overwhelmingly prefer bowling first after winning the toss.
